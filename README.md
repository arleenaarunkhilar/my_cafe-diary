# ☕ Cafe Diary: A Digital Scrapbook

An interactive web application for documenting my cafe experiences. This project uses a functional **Supabase** backend to ensure recommendations are persistent and shareable.

## ✨ Features
*   **Scrapbook Aesthetic:** Designed with tilted polaroid-style cards, washi tape effects, and handwritten typography.
*   **Dynamic Locations:** Users can add specific cities or neighborhoods (e.g., Bhubaneswar, Bangalore) and categorize cafes accordingly.
*   **Real-time Database:** Integrated with **Supabase (PostgreSQL)** to store cafe names, notes, and ratings.
*   **Interactive Ratings:** A custom 5-bean/star rating system for a personalized touch.
*   **Responsive Design:** Styled using **Tailwind CSS** for a seamless experience across mobile and desktop.

## 🛠️ Tech Stack
*   **Frontend:** React.js (via CDN), Tailwind CSS
*   **Backend/Database:** Supabase (PostgreSQL)
*   **Deployment:** GitHub Pages
*   **Fonts:** Quicksand & Indie Flower (Google Fonts)

## 🚀 How it Works
1. **Home:** View a collection of locations I've visited.
2. **Add:** Click the '+' button to log a new cafe with a photo URL, note, and rating.
3. **Persist:** Data is instantly synced to the Supabase cloud, making it visible to anyone with the link.
